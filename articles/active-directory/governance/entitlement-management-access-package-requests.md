# <a name="view-requests-for-an-access-package-in-azure-ad-entitlement-management"></a>在 Azure AD 權利管理中查看存取套件的要求

在 Azure AD 權利管理中，您可以看到誰要求存取套件、其原則和狀態。 本文說明如何查看存取套件的要求。

## <a name="view-requests"></a>檢視要求

**先決條件角色：** 全域管理員、使用者管理員、目錄擁有者、存取套件管理員或存取套件指派管理員

1. 在 Azure 入口網站中按一下 [Azure Active Directory]  ，然後按一下 [身分識別治理]  。

1. 在左側功能表中，按一下 [ **存取套件** ]，然後開啟存取套件。

1. 按一下 [ **要求**]。

1. 按一下特定要求以查看其他詳細資料。

    ![存取套件的要求清單](./media/entitlement-management-access-package-requests/requests-list.png)
