1. 將命令複製到文字編輯器。
2. 視需要在命令中進行下列變更：
    1. 將 `<your-text-analytics-key-here>` 值取代為您的金鑰。
    2. 將要求 URL `<your-text-analytics-endpoint-here>` 的第一個部分取代為您自己的端點 URL。
3. 開啟 [命令提示字元] 視窗。
4. 將文字編輯器中的命令貼到命令提示字元視窗中，然後執行該命令。
