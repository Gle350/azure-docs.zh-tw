# <a name="overview-of-the-fedramp-high-blueprint-sample"></a>FedRAMP 高等級藍圖範例概觀

FedRAMP 高等級藍圖範例使用 [Azure 原則](../../../policy/overview.md)提供治理防護措施，協助您存取特定的 FedRAMP 高等級控制措施。 此藍圖可協助客戶針對任何必須實作 FedRAMP 高等級控制措施的已部署 Azure 架構，部署一組核心原則。

## <a name="control-mapping"></a>控制項對應

控制措施對應區段提供此藍圖中納入的各項原則詳細資料，以及這些原則如何處理 FedRAMP 高等級的各個控制措施。 將資源指派給架構時，Azure 原則會評估不符合已指派原則的資源。 如需詳細資訊，請參閱 [Azure 原則](../../../policy/overview.md)。
