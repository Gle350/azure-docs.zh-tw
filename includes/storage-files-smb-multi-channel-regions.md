| Canada | 加拿大中部，加拿大東部 |
| 法國 | 法國中部 |
| 德國 | 德國中西部 |
| 印度 | 印度中部、印度西部 |
| 日本 | 日本東部、日本西部 |
| 南韓 | 韓國中部、韓國西部 |
| 阿拉伯聯合大公國 | 阿拉伯聯合大公國北部 |
| 英國 | 英國南部，英國西部 |
| 美國 | 美國中北部、美國中南部、美國中西部 |
